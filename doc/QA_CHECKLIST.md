# Minimalist New Tab 发布前 QA 清单

每次发布前按这份清单做最小验收。目标不是覆盖所有浏览器组合，而是保证新标签页的核心体验稳定、轻量、可恢复。

## 运行方式

- 扩展模式：在 `chrome://extensions/` 重新加载已解压扩展，然后打开新标签页。
- 页面模式：直接打开 `newtab.html`，用于快速检查布局和基础交互。

## 基础加载

- 页面能正常打开，无空白页。
- 背景图能加载；加载失败时能回退到可用背景。
- 时间、日期、农历显示正常。
- 搜索框可输入、回车后能跳转到当前搜索引擎。
- 控制台没有阻断页面运行的 JavaScript 错误。

## 搜索

- 默认搜索引擎切换后，搜索框占位文案同步更新。
- 搜索建议可显示；建议接口失败时仍可直接搜索。
- `/关键词` 能进入书签搜索。
- 搜索历史开关生效。
- 清空搜索历史后，历史建议不再出现。
- 输入法组合输入时，回车行为不误触发。

## 书签

- 扩展模式下能同步 Chrome 书签。
- 进入文件夹、返回上级、面包屑导航正常。
- 书签搜索结果能打开正确链接。
- 编辑模式能进入和退出。
- 新增书签、新增文件夹、编辑、删除在扩展模式下可用。
- 页面模式下不可用能力有明确提示，不导致页面崩溃。
- favicon 加载失败时显示 fallback，不出现破图占位。
- 自定义图标上传和清除后显示正常。

## Dock

- Dock 链接能正常打开。
- 桌面端拖拽排序正常。
- 移动端长按拖拽有反馈，不误触打开链接。
- 新增、编辑、删除 Dock 链接正常。
- Dock 在窄窗口不遮挡搜索建议和主要书签内容。

## 设置侧栏

- 打开和关闭侧栏动画正常。
- 保存配置后设置持久化。
- 取消关闭不会保存未保存的临时设置。
- 主题预设切换后页面视觉即时更新。
- 高级外观参数可展开、调整、保存。
- 壁纸来源切换、自定义壁纸、本地图片上传正常。
- 语言切换后主要按钮、设置项、提示文案同步更新。
- 配置导出文件可下载。
- 配置导入后能恢复书签、Dock、样式、壁纸和搜索设置。
- 重置数据有二次确认。

## 响应式布局

至少检查以下宽度：

- 1440px 桌面宽屏。
- 1024px 普通桌面或平板横屏。
- 768px 窄窗口。
- 390px 手机宽度。

检查要点：

- 时间、日期、搜索框不重叠。
- 搜索建议不被书签区或 Dock 裁切。
- 面包屑和操作按钮能换行或收缩。
- 书签卡片文字不溢出。
- 设置侧栏在移动端占满宽度且可滚动。
- 底部 Dock 不遮挡最后一行书签。

## 背景可读性

至少检查三类背景：

- 暗色低细节背景。
- 亮色背景。
- 高细节图片背景。

检查要点：

- 时间和搜索框在三类背景下都清晰。
- 书签标题可读。
- Dock 图标边界清楚。
- 设置侧栏打开后主页面不过度干扰阅读。

## 性能和动效

- 首屏无明显白屏。
- 搜索框 hover、focus 不导致大面积闪烁。
- 设置侧栏关闭后页面滚动和 hover 恢复流畅。
- Dock hover 和拖拽无明显卡顿。
- 开启 `prefers-reduced-motion` 时，大部分动画被压缩或关闭。

## 权限检查

- `manifest.json` 不应无意新增权限。
- 新增 host 权限必须能对应到具体功能。
- 涉及 `history`、`topSites`、网络 API 的功能必须在 README 中说明原因。

## 文档检查

- README 中的功能说明与当前页面一致。
- 新增配置项说明是否会被导出。
- 新增快捷命令或主题说明有用户可理解的入口。
- `doc/FUTURE_PLAN.md` 中已完成事项应在后续迭代时更新状态。
